WeatherByte is a simple weather dashboard built with Python and Tkinter.  
It uses the OpenWeather API to show real-time weather information for any city.  

## Features
- Enter a city name and view the current temperature, description, and humidity.
- Loading animation when fetching data.
- Error handling for invalid city names.

## How to Run
1. Install Python 3.x on your computer.
2. Install `requests` library if not already installed:
pip install requests

3. Run the file:
python WeatherByte.py
